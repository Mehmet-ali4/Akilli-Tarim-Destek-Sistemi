AKILLI TARIM SİSTEMİ - GEREKLİ KÜTÜPHANELER LİSTESİ

1. axios - API üzerinden hava durumu verilerini çekmek ve frontend-backend iletişimini sağlamak için kullanılır.
2. dotenv - API anahtarları gibi hassas verileri güvenli yönetmek için kullanılır.
3. express - Backend sunucusunu oluşturmak ve API yönlendirmelerini yönetmek için kullanılır.
4. cors - Farklı portlarda çalışan frontend, backend ve AI servisleri arasında kaynak paylaşımına izin vermek için kullanılır.

YAPAY ZEKA VE MAKİNE ÖĞRENMESİ KATMANI
1. tensorflow & keras - Derin öğrenme tabanlı tahmin modelleri oluşturmak ve eğitmek için.
2. numpy - Sensör verilerini ve matematiksel hesaplamaları yapmak için.
3. flask - Python'daki yapay zeka modelini bir web servisi olarak (API) sunmak için.
4. flask-cors - Python servisinin Node.js backend'i ile haberleşebilmesi için.
